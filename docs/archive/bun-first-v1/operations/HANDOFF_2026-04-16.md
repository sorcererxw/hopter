## What is now implemented

### Local dev loop

- `make reset`
  - stops stale listeners on `5173` and `8787`
  - clears `tmp/air`
- `make dev`
  - runs the Bun supervisor
  - starts Vite
  - starts Go hot reload through `air`
  - performs startup port preflight
  - writes machine-readable dev state
  - writes persistent append-only logs outside the repo
  - prints highlighted console logs for humans
- `make verify-live`
  - attaches to the running dev loop
  - waits for `state=ready`
  - checks the Go health endpoint
  - runs a lightweight browser smoke through the Go origin
  - records validation evidence

## Important decisions preserved

- The live local loop is now a first-class system, not a thin shell wrapper.
- AI should trust the file-based state and log plane over terminal scrollback.
- Do not build another local API in front of the dev logs for AI agents.
- `make validate-go-*` remains the proof/evidence lane, not the default inner loop.
- `AGENTS.md` is the correct top-level agent index for the dev loop.
